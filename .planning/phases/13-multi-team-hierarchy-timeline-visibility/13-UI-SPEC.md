## Phase-Specific UI Patterns

### Global Sub-Team Switcher (Admin Only)

**Location:** Sidebar, below nav items, above user section (per D-09 decision)

**Component structure:**

-   Dropdown/select component with sub-team name display
-   Chevron icon indicating expandability
-   When expanded: list of all sub-teams with radio selection
-   Selected sub-team highlighted with primary-500 background
-   "All Teams" option at top (shows org-wide data)

**Interaction:**

-   Click to expand dropdown
-   Click sub-team name to select and collapse
-   Selection persists via localStorage (key: `selectedSubTeam`)
-   On selection, all pages (dashboard, timeline, performance, projects) auto-refresh with filtered data

**Styling:**

-   Container: px-3 py-2, bg-gray-800/50, border border-gray-700, rounded-lg
-   Selected item: bg-primary-500/20 text-primary-400
-   Unselected item: text-gray-400 hover:text-gray-200 hover:bg-gray-700
-   Typography: text-sm font-medium

### Sub-Team Management Section (/team Page)

**Location:** New tab/section in existing /team page (per D-05, D-06 decisions)

**Layout:**

-   Tabs: "Members" (existing), "Sub-Teams" (new)
-   Sub-Teams tab content:
    -   Header: "Sub-Teams" + "Create Sub-Team" button (admin only)
    -   Grid of sub-team cards:
        -   Sub-team name
        -   Supervisor name (or "No supervisor assigned")
        -   Member count
        -   Actions: Rename, Delete, Assign Supervisor (admin only)
    -   Empty state: "No sub-teams yet" with CTA

**Card styling:**

-   bg-gray-900 border border-gray-800 rounded-xl p-4
-   Hover: border-gray-700
-   Title: text-white font-semibold text-base
-   Supervisor: text-gray-400 text-sm
-   Member count: text-gray-500 text-xs

### Timeline Empty State (Member with No Tasks)

**Per Claude's Discretion area:** Timeline empty state for member with zero assigned tasks

**Content:**

-   Heading: "No tasks assigned"
-   Body: "You don't have any tasks assigned yet. Ask your supervisor to assign you tasks to see them here."
-   Icon: CheckSquare (gray-600)
-   Background: card component
