## Dashboard UI & Layout (DASH)

### DASH-01: Professional dashboard layout with visual hierarchy
**Test Scenario:** Login as any user and view the dashboard at `/`. Verify the layout has clear section groupings, visual hierarchy, and data visualizations instead of raw-number-only stat cards.

**Expected Outcome:** Dashboard sections are visually distinct with clear headings, proper spacing, and data presented in charts/cards rather than plain numbers.

**Automated Coverage:** 33-02 (member), 33-03 (supervisor)

**Status:** pending

---

### DASH-02: Role-appropriate section rendering
**Test Scenario:** 
1. Login as a member - verify team health and KPI sections are NOT visible
2. Login as a supervisor/assistant_manager/manager - verify all sections ARE visible

**Expected Outcome:** Members see only tasks + activity feed. Supervisors/managers see tasks + team health + KPI + activity feed.

**Automated Coverage:** 33-02 (member), 33-03 (supervisor)

**Status:** pending

---

### DASH-03: Mobile responsiveness
**Test Scenario:** Open dashboard in browser DevTools with mobile viewport (375x667). Verify sections stack vertically, all content is accessible, and touch targets are usable.

**Expected Outcome:** Dashboard is usable on mobile with same role-conditional structure as desktop.

**Automated Coverage:** None (manual test required)

**Status:** pending

---

## My Tasks & Deadlines (TASKS)

### TASKS-01: Tasks sorted by urgency
**Test Scenario:** View dashboard my tasks section. Verify overdue tasks appear first, followed by tasks due soon, then other tasks.

**Expected Outcome:** Tasks are sorted: overdue first, then ascending due date.

**Automated Coverage:** 33-01 (backend test verifies sort order)

**Status:** pending

---

### TASKS-02: Overdue/due-soon visual flags
**Test Scenario:**
1. Find an overdue task - verify it has red styling
2. Find a task due within 48h - verify it has a due-soon indicator

**Expected Outcome:** Overdue tasks are visually red. Due-soon tasks show indicator.

**Automated Coverage:** 33-02

**Status:** pending

---

### TASKS-03: Navigate to /tasks from dashboard
**Test Scenario:** Click on a task card in the dashboard. Verify navigation to `/tasks` with that task selected.

**Expected Outcome:** Clicking a task card navigates to the full task view at `/tasks`.

**Automated Coverage:** 33-02

**Status:** pending

---

**Automated Coverage:** 33-03

**Status:** pending

**Automated Coverage:** 33-03

**Status:** pending

---

### HEALTH-03: Link to /performance from health panel

**Expected Outcome:** Team health panel links to full performance page.

**Automated Coverage:** 33-03

**Status:** pending

### FEED-01: Activity feed for all roles
**Test Scenario:** Login as any role. Verify activity feed section is visible and shows recent standup posts.

**Expected Outcome:** All roles see a scoped activity feed of recent standup posts.

**Status:** pending

### FEED-03: Feed item details and navigation

**Status:** pending
