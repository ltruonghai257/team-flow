<task type="auto">
  <name>Task 1: Add Sprint model and update Task/Milestone models</name>
  <files>backend/app/models.py</files>
  <read_first>backend/app/models.py</read_first>
  <action>
1. Add a new `SprintStatus` enum (planned, active, closed).
2. Create the `Sprint` model class with `__tablename__ = "sprints"`. Add columns:
   - `id` (Integer, primary_key, index)
   - `name` (String, nullable=False)
   - `start_date` (DateTime, nullable=True)
   - `end_date` (DateTime, nullable=True)
   - `status` (Enum(SprintStatus), default=SprintStatus.planned)
   - `milestone_id` (Integer, ForeignKey("milestones.id"), nullable=False)
   - `created_at` (DateTime, default=datetime.now(timezone.utc))
3. Add relationships to `Sprint`: `milestone` (back_populates="sprints") and `tasks` (back_populates="sprint").
4. In the `Milestone` model, add `sprints = relationship("Sprint", back_populates="milestone")`.
5. In the `Task` model, add `sprint_id = Column(Integer, ForeignKey("sprints.id"), nullable=True)`.
6. In the `Task` model, add `sprint = relationship("Sprint", back_populates="tasks")`.
  </action>
  <acceptance_criteria>
    - `grep "class Sprint(Base):" backend/app/models.py` must return a match

    - `grep "SprintStatus" backend/app/models.py` must return a match
  </acceptance_criteria>
</task>
