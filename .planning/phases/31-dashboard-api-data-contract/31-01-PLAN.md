**Add a new dataclass `KpiComputeResult`:**
```python
from dataclasses import dataclass, field
from typing import Any

@dataclass
class KpiPerMemberData:
    user_id: int
    full_name: str
    avatar_url: Optional[str]
    active_tasks: int
    completed_30d: int
    overdue_tasks: int
    kpi_score: int

@dataclass
class KpiComputeResult:
    scorecards: list  # list of KPIMemberScorecard
    per_member: list  # list of KpiPerMemberData
    avg_score: int
    needs_attention_count: int
    completion_rate: float
```

**Add `compute_kpi_overview(db: AsyncSession, sub_team: Optional[SubTeam]) -> KpiComputeResult`:**

This is the async function containing the aggregate SQL query and scoring loop currently in `performance.py` `get_kpi_overview` (lines 452â€“596). Port it verbatim, with two changes:
1. Add `overdue_tasks` aggregate to the per-member query:
   ```python
   func.count(Task.id).filter(
       Task.due_date < now,
       Task.custom_status_id == CustomStatus.id,
       CustomStatus.is_done.is_(False),
   ).label("overdue_tasks"),
   ```
2. After the scoring loop, build `per_member` list from the raw rows + derived kpi_score:
   ```python
   per_member_data.append(KpiPerMemberData(
       user_id=row.id,
       full_name=row.full_name,
       avatar_url=row.avatar_url,
       active_tasks=row.active_tasks,
       completed_30d=row.completed_30d,
       overdue_tasks=row.overdue_tasks,
       kpi_score=kpi_score,
   ))
   ```
3. Compute `completion_rate`:
   ```python
   total_active = sum(m.active_tasks for m in per_member_data)
   total_completed = sum(m.completed_30d for m in per_member_data)
   completion_rate = total_completed / (total_active + total_completed) if (total_active + total_completed) > 0 else 0.0
   ```
4. Return `KpiComputeResult(scorecards=scorecards, per_member=per_member_data, avg_score=avg_score, needs_attention_count=len(needs_attention), completion_rate=round(completion_rate, 4))`
